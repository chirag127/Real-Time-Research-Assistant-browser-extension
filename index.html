About

Features

Topic Detection

Automatically identifies research topics from your browsing content.

Content Suggestions

Recommends relevant articles, papers, and reports in real-time.

Summarization

Provides AI-generated summaries to quickly grasp key points.

Save Information

Save articles, quotes, and key points for later reference.

Customization

Personalize content preferences and research areas.

Cross-Browser Support

Works seamlessly on Chrome, Edge, and Firefox.

Installation

  1. Download or clone the repository from GitHub.
  2. Open your browser's Extensions page.
  3. Enable Developer Mode.
  4. Click "Load unpacked" and select the extension/ folder.
  5. Enjoy enhanced research capabilities!

How to Use

While browsing, the extension will detect topics and suggest relevant content. Access summaries and save important information directly from the extension popup.

Technology Stack

  • JavaScript, HTML, CSS
  • Chrome Extension APIs (Manifest V3)
  • Express.js Backend
  • Gemini 2.0 Flash Lite AI Model
